## 📱 安卓应用打包

### 快速打包（3步完成）

1. **检查JDK**
   ```
   双击：check_jdk.bat
   ```
   - 如果JDK未安装，按照提示安装

2. **构建应用**
   ```
   双击：build_final.bat
   ```

3. **安装APK**
   ```
   文件位置：platforms\android\app\build\outputs\apk\debug\app-debug.apk
   ```

### 详细指南

查看 `BUILD_GUIDE.md` 或 `BUILD_SIMPLE.txt` 获取完整的打包说明

## 📱 手机访问（网页版）

### 局域网访问（同一WiFi）

1. **启动服务器**
   - 双击 `start_server.bat`
   - 服务器启动后会显示局域网地址

2. **手机连接**
   - 确保手机和电脑连接同一WiFi
   - 打开手机浏览器
   - 输入服务器显示的局域网地址
   - 例如：`http://192.168.1.100:3000`

### 查看手机访问指南

详细指南请查看 `手机访问指南.txt` 文件

## 📋 项目简介

"战争前线"是一款完全仿照KARDS机制开发的二战题材战略卡牌对战游戏。

### 核心特色

- ✅ **5个国家阵营**：德国🇩🇪、美国🇺🇸、英国🇬🇧、苏联🇷🇺、日本🇯🇵
- ✅ **105张卡牌**：涵盖单位、指令、增援等多种类型
- ✅ **完整战斗系统**：前线争夺、指挥点管理、单位移动和攻击
- ✅ **成就系统**：15个成就，激励玩家探索不同玩法
- ✅ **卡背定制**：12个预设卡背+自定义卡背创建
- ✅ **联机对战**：支持房间创建和加入，实现实时对战

## 🎮 游戏操作

### 快捷键

- `Esc` - 取消当前行动
- `E` - 结束回合
- `W` - 待命

### 基本操作

1. **部署单位**：点击手牌 → 点击支援战线空位
2. **移动单位**：点击单位 → 点击"移动单位" → 点击前线空位
3. **攻击**：点击单位 → 点击"攻击目标" → 点击敌方单位
4. **待命**：点击单位 → 点击"待命"
