## 📋 项目概述

"战争前线"是一款完全仿照KARDS（The WWII Card Game）机制开发的二战题材战略卡牌对战游戏。游戏采用回合制战斗系统，玩家通过部署单位、使用指令卡牌来摧毁敌方总部。

### 核心特色

- **5个国家阵营**：德国🇩🇪、美国🇺🇸、英国🇬🇧、苏联🇷🇺、日本🇯🇵
- **105张精心设计的卡牌**：涵盖单位、指令、增援等多种类型
- **完整的战斗系统**：前线争夺、指挥点管理、单位移动和攻击
- **成就系统**：15个成就，激励玩家探索不同玩法
- **卡背定制**：12个预设卡背+自定义卡背创建
- **联机对战支持**：支持房间创建和加入，实现实时对战

## 🎮 游戏机制

### 战场布局

```
┌─────────────────────────────────┐
│  敌方支援战线 (4格)              │
├─────────────────────────────────┤
│  前线 (4格) - 可争夺控制权       │
├─────────────────────────────────┤
│  己方支援战线 (4格)              │
├─────────────────────────────────┤
│  手牌区 (最多10张)               │
│  指挥点显示                      │
└─────────────────────────────────┘
```

### 回合流程

1. **回合开始**
   - 增加指挥点（最多12点）
   - 抽1张卡牌
   - 重置所有单位状态

2. **部署阶段**
   - 消耗指挥点部署单位
   - 只能部署到支援战线

3. **行动阶段**
   - 移动单位（支援→前线）
   - 攻击敌方单位/总部
   - 使用指令卡牌

4. **回合结束**
   - 清除临时效果
   - 检查游戏结束条件

### 卡牌类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 单位卡 | 部署到战场的作战单位 | 步兵师、虎式坦克 |
| 指令卡 | 立即生效的效果 | 闪电战指令、后勤支援 |
| 增援卡 | 直接获得单位 | 空降兵、神风特攻队 |
| 压制卡 | 降低敌方单位状态 | 防空警报、突袭 |
| 支援卡 | 增强己方单位效果 | 装甲增援、空军优势 |
| 反击卡 | 被动触发效果 | 反坦克炮 |

### 关键字系统

| 关键字 | 效果 |
|--------|------|
| 突击 (BLITZ) | 攻击时不受到反击伤害 |
| 反坦克 (ANTI_TANK) | 攻击坦克时伤害翻倍 |
| 对空 (ANTI_AIR) | 攻击飞行单位时伤害翻倍 |
| 装甲 (ARMOR) | 减少受到的伤害 |
| 轰炸 (BOMBARDMENT) | 可从任意位置攻击 |
| 先攻 (FIRST_STRIKE) | 在敌人反击前造成伤害 |
| 防守者 (DEFENDER) | 被动时+1护甲 |
| 支援 (SUPPORT) | 提供支援效果给相邻单位 |
| 烟雾 (SMOKE) | 无法被直接攻击 |
| 潜行 (STEALTH) | 攻击前无法被选中 |
| 冲锋 (RUSH) | 部署后可以立即行动 |
| 坚守 (HOLD) | 前线单位死亡时移至前线 |
| 接力 (RELAY) | 死亡时让相邻单位行动 |

## 🐛 已修复的Bug

### Bug #1：回合数计算错误
- **问题**：回合数只在玩家1回合时增加
- **修复**：每次切换玩家时都增加回合数

### Bug #2：指挥点计算错误
- **问题**：指挥点增长不正确
- **修复**：正确根据回合数计算最大指挥点

### Bug #3：单位HP未定义时的默认值
- **问题**：某些单位HP未定义时导致错误
- **修复**：添加默认值处理

### Bug #4：攻击总部时未考虑装甲属性
- **问题**：攻击总部时装甲属性未生效
- **修复**：正确应用装甲减伤

### Bug #5：部署时缺少阵营验证
- **问题**：可以部署其他阵营的卡牌
- **修复**：添加阵营匹配检查

### Bug #6：单位移动/攻击次数未定义
- **问题**：部分单位的移动/攻击次数未定义
- **修复**：添加默认值（移动1次，攻击1次）

### Bug #7：指令卡执行失败时未退还资源
- **问题**：指令卡执行失败时，指挥点不退还
- **修复**：添加错误处理和资源退还

## ⚖️ 卡牌平衡性优化

### 已优化的卡牌（5张）

#### 1. 战时生产
- **原属性**：1费，获得3点指挥点
- **优化后**：2费，获得2点指挥点
- **原因**：过于超模，1费获得3点指挥点严重影响游戏平衡

#### 2. 空降兵
- **原属性**：2费，部署2个单位+抽1张牌
- **优化后**：4费，部署1个单位到前线
- **原因**：性价比过高，2费就获得2个单位+抽牌

#### 3. 第101空降师
- **原属性**：4费，部署3个单位+抽1张牌
- **优化后**：7费，部署2个单位（稀有度提升）
- **原因**：过于超模，4费获得3个单位+抽牌

#### 4. 乌拉冲锋
- **原属性**：2费，部署3个单位+抽2张牌
- **优化后**：4费，部署2个单位
- **原因**：极其超模，2费就获得大量资源和单位

#### 5. 人海战术
- **原属性**：3费，部署4个单位+抽2张牌
- **优化后**：6费，部署3个单位+抽1张牌（稀有度提升）
- **原因**：过于超模，3费就能获得巨大优势

## 🎮 游戏操作指南

### 基本操作

- **部署单位**：点击手牌 → 点击支援战线空位
- **移动单位**：点击单位 → 点击"🚶 移动单位" → 点击前线空位
- **攻击**：点击单位 → 点击"⚔️ 攻击目标" → 点击敌方单位
- **待命**：点击单位 → 点击"⏸️ 待命"
- **结束回合**：点击"结束回合"按钮

### 键盘快捷键

- `Esc` - 取消当前行动
- `E` - 结束回合
- `W` - 待命

### 牌组构建

1. 点击主菜单的"📦 构建牌组"
2. 选择主阵营（必需）和次阵营（可选）
3. 从可用卡牌中选择卡牌添加到牌组
4. 添加40张卡牌后保存牌组

### 成就系统

1. 点击主菜单的"🏆 成就"
2. 查看所有成就和完成条件
3. 成就解锁后自动获得金币奖励

### 卡背定制

1. 点击主菜单的"🎨 卡背定制"
2. 从预设卡背中选择或创建自定义卡背
3. 部分卡背需要通过成就解锁
