# 战争前线 - 手机应用指南

## 📱 概述

战争前线是一个基于 Web 技术的卡牌对战游戏，现在已经支持打包成 Android 手机应用。本指南将帮助你：

1. 将游戏打包成 Android APK
2. 在手机上安装和运行
3. 设置手机与手机之间的联机对战

---

## 🔧 环境准备

### 方法1: 自动设置（推荐）

1. **运行环境设置脚本**
   ```
   双击运行: setup_environment.bat
   ```
   这个脚本会自动检查并安装所有必需的组件。

2. **等待安装完成**
   - 脚本会自动安装 Node.js、Cordova、Android SDK 等
   - 首次安装可能需要 10-20 分钟

### 方法2: 手动安装

如果你更喜欢手动安装，请按照以下步骤：

1. **安装 Node.js**
   - 访问: https://nodejs.org/
   - 下载并安装 Node.js 18 或更高版本

2. **安装 Java JDK**
   - 访问: https://www.oracle.com/java/technologies/downloads/#java17
   - 下载并安装 JDK 17 或更高版本

3. **安装 Android Studio**
   - 访问: https://developer.android.com/studio
   - 下载并安装 Android Studio
   - 打开 Android Studio，进入 SDK Manager
   - 安装 Android SDK Platform-Tools
   - 安装 Android SDK Build-Tools
   - 安装至少一个 Android SDK Platform (推荐 API 33)

4. **安装 Cordova**
   ```bash
   npm install -g cordova
   ```

---

## 📦 构建 APK

### 快速构建

1. **运行构建脚本**
   ```
   双击运行: build_simple.bat
   ```

2. **选择构建类型**
   - 输入 `1` 构建 Debug 版本（用于测试）
   - 输入 `2` 构建 Release 版本（用于发布）

3. **等待构建完成**
   - 构建过程可能需要 5-15 分钟
   - 构建完成后，APK 文件位于:
     ```
     platforms\android\app\build\outputs\apk\debug\app-debug.apk
     ```

### 详细构建（可选）

如果你需要更多控制，可以使用以下命令：

```bash
# 构建 Debug 版本
cordova build android

# 构建 Release 版本
cordova build android --release

# 在设备上运行
cordova run android
```

## 📲 安装到手机

### 方法1: 直接安装 APK

1. **传输 APK 文件到手机**
   - 通过 USB 数据线连接手机和电脑
   - 将 APK 文件复制到手机存储

2. **在手机上安装**
   - 打开手机文件管理器
   - 找到 APK 文件
   - 点击安装
   - 如果提示"允许安装未知来源应用"，请允许

3. **启动应用**
   - 安装完成后，点击"打开"
   - 或者在应用列表中找到"War Front"

### 方法2: 使用 ADB 安装

如果你已经安装了 Android SDK Platform-Tools：

```bash
# 连接手机（需要启用 USB 调试）
adb devices

# 安装 APK
adb install platforms\android\app\build\outputs\apk\debug\app-debug.apk

# 卸载应用
adb uninstall com.warfront.game
```

## 🌐 手机联机对战

### 方式1: 局域网对战（同一 WiFi）

**准备阶段:**

1. **准备两台手机**
   - 确保两台手机都安装了游戏应用
   - 确保两台手机连接到同一个 WiFi 网络

2. **准备一台电脑作为服务器**
   - 在电脑上运行游戏服务器
   - 双击运行: `启动服务器.bat`
   - 记录显示的 IP 地址，例如: `192.168.1.100`

**连接阶段:**

1. **在手机1上**
   - 打开游戏应用
   - 点击"联机对战"
   - 在"服务器地址"输入框中输入: `http://192.168.1.100:3000`
   - 点击"连接服务器"
   - 输入玩家名称
   - 点击"创建房间"
   - 记录房间 ID，例如: `ABC123`

2. **在手机2上**
   - 打开游戏应用
   - 点击"联机对战"
   - 在"服务器地址"输入框中输入: `http://192.168.1.100:3000`
   - 点击"连接服务器"
   - 输入玩家名称
   - 在"房间ID"输入框中输入: `ABC123`
   - 点击"加入房间"

3. **开始游戏**
   - 两台手机都会显示房间信息
   - 每个玩家选择牌组
   - 点击"准备"
   - 游戏自动开始

### 方式2: 手机直接对战（无需电脑）

如果你希望两台手机直接对战，无需电脑服务器：

1. **在一台手机上启动服务器**
   - 需要使用特殊版本的应用（包含服务器功能）
   - 或使用 Web 服务器应用

2. **获取手机 IP 地址**
   - 打开手机设置
   - 进入 WiFi 设置
   - 查看当前连接的 IP 地址

3. **另一台手机连接**
   - 输入第一台手机的 IP 地址
   - 按照上述步骤连接和游戏

### 方式3: 使用云服务器

如果你想随时随地与朋友对战：

1. **购买云服务器**
   - 推荐: 阿里云、腾讯云、AWS 等
   - 选择最低配置即可（1核1G）

2. **部署游戏服务器**
   - 上传服务器代码到云服务器
   - 安装 Node.js
   - 运行 `node server/server.js`

3. **配置端口**
   - 开放 3000 端口
   - 或使用 Nginx 反向代理

4. **手机连接**
   - 输入云服务器的公网 IP 地址
   - 例如: `http://your-server-ip:3000`

## ⚙️ 游戏设置

### 服务器地址配置

1. **手动输入**
   - 在联机对战界面
   - 输入服务器地址
   - 点击"连接服务器"

2. **使用预设地址**
   - 点击"本地服务器"按钮
   - 自动填充 `http://localhost:3000`

3. **自动检测**
   - 点击"自动检测"按钮
   - 应用会自动扫描局域网中的服务器

4. **使用历史地址**
   - 应用会保存最近使用的服务器地址
   - 在"最近使用的服务器"列表中选择

### 玩家设置

- **玩家名称**: 在联机对战界面输入你的名称
- **牌组选择**: 先在"构建牌组"中创建牌组
- **游戏设置**: 在"设置"中调整音量、动画等

## 🎮 游戏操作

### 基本操作

- **部署单位**: 点击手牌 → 点击支援战线空位
- **移动单位**: 点击单位 → 点击"移动单位" → 点击前线空位
- **攻击**: 点击单位 → 点击"攻击目标" → 点击敌方单位
- **待命**: 点击单位 → 点击"待命"
- **结束回合**: 点击"结束回合"按钮

### 触摸优化

- **放大触摸目标**: 所有按钮都针对触摸进行了优化
- **触摸反馈**: 点击时会有视觉反馈
- **防误触**: 重要操作需要确认

### 横屏模式

- 游戏推荐横屏游玩
- 竖屏时会显示提示
- 在设置中可以锁定横屏

## 🔍 故障排除

### 无法连接服务器

**问题**: 点击"连接服务器"后显示连接失败

**解决方案**:
1. 检查服务器是否正在运行
2. 检查手机和服务器是否在同一网络
3. 检查服务器地址是否正确
4. 检查防火墙是否阻止连接
5. 尝试使用 IP 地址而不是 localhost

### 游戏卡顿

**问题**: 游戏运行缓慢或卡顿

**解决方案**:
1. 关闭其他后台应用
2. 在设置中关闭动画效果
3. 降低音量和音效
4. 重启游戏

### APK 无法安装

**问题**: 安装 APK 时提示错误

**解决方案**:
1. 允许安装未知来源应用
2. 检查 Android 版本（需要 5.0 或更高）
3. 卸载旧版本后再安装
4. 确保手机有足够的存储空间

### 游戏显示不全

**问题**: 游戏界面显示不完整

**解决方案**:
1. 使用横屏模式
2. 调整手机显示设置
3. 在设置中调整字体大小

### 构建失败

**问题**: 运行构建脚本时失败

**解决方案**:
1. 运行 `setup_environment.bat` 重新设置环境
2. 检查 Java 版本（需要 JDK 17+）
3. 检查网络连接
4. 清理构建文件:
   ```bash
   cd platforms\android
   gradlew.bat clean
   ```

## 📝 技术信息

### 应用信息

- **应用名称**: War Front
- **包名**: com.warfront.game
- **版本**: 0.2.0
- **最低 Android 版本**: 5.0 (API 21)
- **目标 Android 版本**: 13 (API 33)

- **前端**: HTML5, CSS3, JavaScript (ES6+)
